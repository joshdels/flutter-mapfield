# TopmapField
MapField is an offline mobile GIS app designed for field data collection and management. Also collects and connect to backend api for cloud sync and storage

Built for TopMapSolutions.com, it allows field teams to collect, store, and visualize geographic data without needing an internet connection.

### Features
- Offline first support
- Field data collection and editing

### TechStack
- flutter
- riverpods - state management
- sqlite - local database
- api backend - django(topmapsolutions)


### Getting Started
```
cd mapfield
flutter pub get
```
for windows/linux
```
flutter run -d windows
flutter run -d linux
```
